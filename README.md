# Lógica de Programação e Manipulação de Dados para Administração

Este repositório contém uma série de exercícios práticos desenvolvidos para aplicar lógica de programação e análise de dados em cenários reais da área de administração, como controle de vendas, margem de lucro e gestão de estoque.

## 🎯 Objetivos do Projeto
O objetivo principal é praticar a resolução de problemas de gestão utilizando Python, focando em:
- Estruturas condicionais para tomada de decisão.
- Laços de repetição para processamento de listas.
- Manipulação de grandes volumes de dados com a biblioteca Pandas.
- Visualização de indicadores de desempenho (KPIs).

## 📊 Conteúdo dos Exercícios
O projeto está dividido em seções práticas:
1. **Lógica e Condicionais:** Classificação de desempenho de vendas e análise de margem de lucro.
2. **Comandos Iterativos:** Cálculo de médias de vendas diárias e contagem de estoque/devoluções.
3. **Pandas e Arquivos:** Leitura de relatórios em Excel (`.xlsx`) e filtragem de dados específicos.
4. **Manipulação de Dados:** Criação de métricas como "Receita Total" e "Ticket Médio" (Valor por Item).
5. **Visualização de Dados:** Geração de gráficos de barras e linhas utilizando Seaborn e Matplotlib para análise temporal e por produto.

## 🛠️ Tecnologias Utilizadas
- **Python 3**
- **Pandas:** Para manipulação e análise de dados estruturados.
- **Seaborn & Matplotlib:** Para criação de gráficos estatísticos.
- **Jupyter Notebook / Google Colab:** Ambiente de desenvolvimento.
